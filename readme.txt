- Cadastrar;
- Editar;
- Excluir;
- Pesquisar os discos.

Os dados do Discos são:
- Título;
- Album;
- Capa (Upload de arquivo).

Dashboard com resultados para o login funcionário e Master visualizar (Dia/Mês/Ano)
- Número de discos adicionados;
- Categorias listadas;
- Autores listados.
